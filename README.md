# IT Automation Toolkit

## 🚀 Overview
This project contains practical automation scripts designed for IT Support and Application Support environments.

It focuses on real-world tasks such as user management, ticket handling, and system monitoring.

## 🧰 Tech Stack
- Python
- REST APIs (requests)
- Logging

## ⚡ Features
- User provisioning simulation
- Ticket analysis from CSV data
- Log file parsing and error detection

## 📂 Project Structure
- user_provisioning.py → simulate user lifecycle management
- ticket_analyzer.py → analyze support tickets
- log_parser.py → detect errors in system logs

## 🎯 Why this matters
This project reflects how IT Support Engineers can automate repetitive tasks and improve operational efficiency.
